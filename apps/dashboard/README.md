## Quadra EDMS Dashboard

The beautiful dashboard for managing construction projects, documents, workflows, and transmittals.

### Features

- **Project Management**: Track construction projects and their status
- **Document Control**: Manage project documents with version control
- **Workflows**: Automate document approval processes
- **Transmittals**: Send and track document transmittals
- **Financial Tracking**: Invoices, transactions, and reporting
- **Team Collaboration**: Multi-user support with role-based access

### Tech Stack

- **Framework**: Next.js 16 with Turbopack
- **Auth**: Better Auth
- **Database**: Turso (SQLite)
- **ORM**: Drizzle
- **UI**: Midday UI components (rebranded for Quadra)
- **Styling**: Tailwind CSS

### Getting Started

```bash
bun install
bun run dev
```

Open [http://localhost:3001](http://localhost:3001) to see the dashboard.
