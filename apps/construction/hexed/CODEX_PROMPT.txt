## KNOWN ISSUES & SOLUTIONS

### Issue 1: Dashboard Access (CRITICAL)
**Problem:** Session doesn't refresh after profile update
**Solution:** Implement session refresh in actions/users.ts
**Reference:** AGENT.md "PERMANENT FIX" section

### Issue 2: Vercel Blob Token
**Problem:** BLOB_READ_WRITE_TOKEN may not be set
**Solution:** Create token in Vercel dashboard → Storage → Blob
**Impact:** Document uploads will fail without this

### Issue 3: Resend Email
**Problem:** RESEND_FROM_EMAIL may not be verified
**Solution:** Verify sender email in Resend dashboard
**Impact:** Email notifications will fail without this

### Issue 4: OAuth Redirect URLs
**Problem:** OAuth callbacks may not include production URL
**Solution:** Add production URL to Google/GitHub OAuth app settings
**Impact:** Login will fail in production without this
