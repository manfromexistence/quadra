================================================================================
CODE STYLE HIGHLIGHTS (see dx.md for full details)
================================================================================

React Patterns:
- Avoid multiple useState - use state machines/reducers
- Prefer derived state over useEffect
- Server components by default, client only when needed
- Extract complex conditionals into components

TypeScript:
- Use interfaces for objects
- Use maps instead of enums
- Descriptive variable names (isLoading, hasError)
- console.log({ value }) not console.log(value)

Styling:
- Tailwind utility classes
- Use cn() for conditional classes
- Use flex and gap (not space-x-* or space-y-*)
- Mobile-first responsive design

File Naming:
- Directories: lowercase-with-dashes
- Components: PascalCase.tsx
- Utilities: camelCase.ts
